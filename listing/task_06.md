Что выведет программа? Объяснить вывод программы. Рассказать про внутреннее устройство слайсов и что происходит при передачи их в качестве аргументов функции.

```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Будет выведено: [3 2 3]

До изменения массива в слайсе в функции изменения будут отражаться на исходном массиве, 
на который указывают оба слайса. Однако после расширения слайса внутри функции, 
слайс будет указывать на новую область памяти. Это приведет к тому, что изменения, 
происходящие после расширения, больше не будут влиять на слайс в main.

Слайс в Go состоит из трех компонентов: указателя на массив, длины и емкости. 
Длина слайса — это количество элементов, которые в нем содержатся, а емкость — это максимальное количество элементов, 
которое слайс может вместить, не выделяя новый массив. При передаче слайса в функцию в Go создается его копия, 
что означает передачу копий указателя на массив, длины и емкости. Из-за этого, когда слайс изменяется внутри функции 
(например, расширяется с помощью append), он начинает указывать на новый массив, и изменения не влияют на оригинальный слайс в main.
```
